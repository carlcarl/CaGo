# CaGo #

A web page to parse and display go gibo(圍棋棋譜) 

## Info ##

* File type: sgf
* Size: 19X19

## Usage ##

In index.html, find *findPath* and modify it.

	var filePath = "./example.sgf";
	// var filePath = "whatyouwant.sgf";
