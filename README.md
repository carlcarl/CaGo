# CaGo #

A web page to parse and play go gibo(圍棋棋譜) 

## Requirement ##

* File type: sgf
* Size: 19X19

## Usage ##

In index.html, find *cago.go* and modify its argument.

	cago.go("./sample.sgf");
	// cago.go("./whatyouwant.sgf");
